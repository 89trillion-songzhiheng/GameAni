# GameAni

**整体大纲**
1.设计我方弓箭手与敌方弓箭手
2.设计弓箭飞行特效
3.设计人物移动脚本，伤害计算脚本

**目录结构**。

│   ├── ArmyModel.cs  //json文档实体类
│   ├── ArmyModel.cs.meta

**界面结构**
 Hierarchy：
    1.Enemy  //敌人
    2.Shooter //弓箭手
    3.Canvas //展示血量的画布
      1）ShooterBlood //射手血量
      2）EnemyBlood //敌人血量
